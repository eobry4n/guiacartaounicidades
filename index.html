Guia de Vendas – Cartões de Alta Renda

Como usar este guia
  • Verifique o limite: O limite mínimo é obrigatório para emissão.
  • Analise os critérios: Para Black/Infinite e Merit, comprove renda OU patrimônio (investimentos + saldo médio 3m + capital social). Para o Zenith, todos os critérios são obrigatórios (limite, renda e patrimônio).
  • Benefícios: Use os benefícios para vincular ao perfil (viagens, pontos, lounges).
  • Isenção: Programa Comprou Zerou.
